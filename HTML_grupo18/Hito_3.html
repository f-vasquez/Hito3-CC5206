Introducción¶

El informe que se exhibe a continuacin trata sobre el anlisis y el trabajo realizado sobre un conjunto de datos concerniente al modelo de los Cinco Grandes rasgos de personalidad (Big Five Personality traits). El dataset contiene informacin sobre 50 preguntas correspondientes a 5 grandes dominios de personalidad.

El objetivo principal del proyecto consiste en analizar el conjunto de datos escogido, encontrar clusters dentro de las respuestas y asociarlos a cierto tipo de personalidad para luego validar o no la categorizacin de personalidades por grupos.

Contexto general¶

Contexto¶

Los Cinco Grandes rasgos de personalidad es el modelo de personalidad ms aceptado y utilizado en psicologa acadmica. Proviene del estudio estadstico de las respuestas a tems de personalidad.

Utilizando una tcnica llamada anlisis factorial, investigadores observaron las respuestas de las personas a cientos de elementos de personalidad e hicieron la pregunta

"Cul es la mejor manera de resumir a un individuo?"

Esto se ha realizado con muchas muestras alrededor del mundo y el resultado general es que, si bien parece haber variables de personalidad ilimitadas, cinco se destacan del paquete en trminos de explicar las respuestas de muchas personas a preguntas sobre su personalidad: extraversin (extraversion), neuroticismo o inestabilidad emocional (neuroticism), amabilidad (agreeableness), meticulosidad (conscientiousness) y actitud receptiva hacia la experiencia (openness to experience).

El dataset a estudiar se recopil (2016-2018) a travs de un test de personalidad en lnea. Dicho test usa los marcadores de factores Big-Five del IPIP (International Personality Item Pool), desarrollado por Goldberg (1992).

Motivación¶

Analizar el conjunto de datos escogido, con el propsito de extraer conocimiento, especficamente encontrar clusters dentro de las respuestas y asociarlos a cierto tipo de personalidad, buscando validar las categoras de personalidades propuestas.

Tambin, identificar patrones previamente desconocidos, agregando valor a los datos para as ser capaces obtener otras conclusiones adems de las personalidades.

Objetivos principales¶

El objetivo del test consiste en medir de cierta forma los 5 aspectos de la personalidad descritos por el modelo Big Five, por lo que la primera pregunta que surge naturalmente y a modo de problemtica es si es posible comparar las personalidades de distintos individuos en base a esta mtrica.

De aqu se desprende el objetivo principal de este proyecto, que es validar o no la categorizacin de personalidades por grupos usando la mtrica provista por el test aplicado sobre un conjunto de individuos.

Otras aspectos interesantes a analizar conforman los objetivos secundarios del proyecto, tales como:

  • Evaluar la influencia de la cultura sobre la personalidad y/o grupos de personalidades. Con esto se pretende comparar los efectos de la cultura de un pas o una regin sobre las personalidades de la poblacin y reconocer ciertas tendencias.
  • Dada la falta de categorizacin en la base de datos a utilizar, se buscar encontrar la cantidad de clusters que explica de mejor manera los datos, contrastando con las cantidades de personalidades existentes en distintas referencias en la literatura.

Descripción¶

El dataset estudiado cuenta con 110 variables distintas:

  • 50 preguntas relacionadas a las reas Extraversin, Amabilidad, Escrupulosidad, Estabilidad Emocional e Intelecto (10 de cada una), respondidas en la escala Likert (1 - 5), correspondiendo entonces a una variable cualitativa del tipo ordinal.

  • 50 variables son el tiempo en milisegundos que tom contestar cada una de esas preguntas, por lo tanto, correspondiendo a una variable cuantitativa del tipo razn.

    • introelapse, testelapse y endelapse: tiempos en segundos en la pantalla de inicio, preguntas y de finalizacin de la encuesta, variables de tipo cuantitativa y de razn.

    • IPC: nmero de respuestas en esta direccin IP, variable cuantitativa y de razn.

    • country: pas, determinado por informacin tcnica, cualitativa y nominal.

    • lat_appx_lots_of_err y long_appx_lots_of_err: latitud y longitud aproximada (aunque recomiendan no darle uso), cuantitativa e intervalo.

def PCA_apply(df, plot = True):
    
    pca = PCA(0.8)
    pca_result = pca.fit_transform(df)
    suma = np.cumsum(pca.explained_variance_ratio_)
    pca_data = pca.transform(df)
    
    plt.rcParams['figure.figsize'] = (15,8)

    per_var = np.round(pca.explained_variance_ratio_* 100, decimals=1)
    labels = ['PC' + str(x) for x in range(1, len(per_var)+1)]
    
    pca_df = pd.DataFrame(pca_data, index=df.index, columns=labels)
    
    if plot:
        fig, ax = plt.subplots()
        df_pca_componentes = pd.DataFrame(pca.components_, columns = df.columns).T

        X = list(df_pca_componentes[0])
        Y = list(df_pca_componentes[1])
        nombres = df_pca_componentes.index

        for i, type in enumerate(nombres):
            x = X[i]
            y = Y[i]
            plt.scatter(x, y , marker = 'o')
            plt.text(x,y+0.01,type, fontsize = 17)

        plt.title('Contribucin de Variables a Primeras 2 Componentes',fontsize=30)
        plt.xlabel('PC1 - Var. Exp.: {0}%'.format(per_var[0]),fontsize=25)
        plt.ylabel('PC2 - Var. Exp.: {0}%'.format(per_var[1]),fontsize=25)

        plt.axhline(y = 0, alpha = 1)
        plt.axvline(x = 0, alpha = 1)
        plt.rcParams['figure.figsize'] = (15,8)

        plt.tight_layout()
        plt.show()
    return pca

def get_cluster_count2(query, clf):   
    data = {}
    
    for pais in query:
        
        df = dfpais2[dfpais2['country']==pais].reset_index()    # seleccionar resultados de reino unido
        df = df.drop(['country','index'], axis=1)
        
        dfdata = df.to_numpy()
        predictions = clf.predict(dfdata)     # predecir clases

        class0, class1, class2, class3, class4 = 0,0,0,0,0
        for i in predictions:
            if i==0:    class0+=1
            elif i==1:  class1+=1
            elif i==2:  class2+=1
            elif i==3:  class3+=1
            else:       class4+=1
        
        n=len(predictions)
        data[pais] = [class0/n, class1/n, class2/n, class3/n, class4/n]    # distribicion de clases
    
    return pd.DataFrame(data)
